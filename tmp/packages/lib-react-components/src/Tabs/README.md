# Tabs

Reskins the default Grommet `Tabs` component. All props etc. work in the same way.

Example:

```js
import { Tabs, Tab } from '@zooniverse/react-components'

function Example () {
  return (
    <Tabs>
      <Tab title='Foo'>
        Foo
      </Tab>
      <Tab title='Bar'>
        Bar
      </Tab>
    </Tabs>
  )
}

```
