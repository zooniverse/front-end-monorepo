# Tabs

These are [Grommet's Tab components](https://github.com/grommet/grommet/tree/NEXT/src/js/components/Tabs) reskinned for use in the Classifier.

Usage remains the same:

```js
  import { Tabs, Tab } from './path/to/Tabs';

  <Tabs>
    <Tab title='Tab 1'>...</Tab>
    <Tab title='Tab 2'>...</Tab>
  </Tabs>
```
