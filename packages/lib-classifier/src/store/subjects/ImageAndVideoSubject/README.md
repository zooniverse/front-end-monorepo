# Image(s) And Video(s) Subject

This Subject model represents a Subject that contains 1+ image files and 1+ video files.

Historically, this was used by the [Chimp&See project](https://www.zooniverse.org/projects/sassydumbledore/chimp-and-see), which had 1 mp4 file as its first item (which was a video of a chimp), and 10+ jpeg files (which were stills from the video).
