# SpacedText

Creates the wider-kerned text used in titles etc. Reuses the Grommet `Text` component, and accepts the same props. Additionally, it sets the following props by default:

```js
{
  margin: 'none',
  size: 'small',
  weight: 'normal'
}
```

## Example

```js
<SpacedText>
  Letter-spaced title
</SpacedText>
```
