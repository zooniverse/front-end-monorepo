# Billion Classifications Countdown

The Billion Classifications Countdown component is intended to be a **temporary** addition to the Stats component, to celebrate the billion classifications milestone.

Features:

- Displays number of classifications to reach 1,000,000,000.
  - Uses our Eras stats service.
  - Once classifications exceeds 1,000,000,000 + some buffer, this countdown will stop appearing on the home page.

Dev Notes:

- This component was created during the 2026 May Hackday and intended to go up _before_ we reach 1,000,000,000 classifications, obviously.
